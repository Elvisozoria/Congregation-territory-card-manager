# Territory Cards — V2 (No Installation Required)

## How to use

1. Download this folder
2. Open `index.html` in your browser
3. Done!

## Saving your data

- Click **Save JSON** to download your territories as a file
- Click **Load JSON** to restore from a saved file
- **Tip:** Save your JSON file in your Google Drive or Dropbox folder so it backs up automatically

## Importing from Google Earth

1. Export your territories as KML or KMZ from Google Earth
2. Click **Import KML** and select the file
3. Territories will be added automatically

## Features

- Draw territory boundaries on an interactive map
- Add landmarks (points of reference) to each territory
- Generate printable territory cards with QR codes
- Download cards as PNG images
- Print all cards at once
